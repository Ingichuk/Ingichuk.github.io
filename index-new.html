소개

활빈당에는 훌륭한 경험을 가진 개발자들이 있습니다
저희는 스타트업이 겪고 있는 기술부채에 대해서
매우 깊이 이해하고 있습니다.
확장가능하고 효율적이고 안전한 서비스를 위해서
각 분야의 개발 전문가들의 도움을 받아보세요

서비스

  • Scalable : 현재 사용자 규모에 적합하고, 확장이 용이한 아키텍쳐를 안내해드립니다
  • Data Management : 빅데이터를 위해서 각 기업 수준에 맞는 데이터 관리 방법을 알려드립니다
  • Security : 보안에 취약한 스타트업에 환경에서 최소한의 노력으로 최대의 보안을 유지할 수 있도록 합니다
  • DevOps : 최소한으로 갖춰야할 프로덕트 개발 방법론, 툴에 대해서 알려드립니다
  • iOS/Android Deep-down : 네이티브 앱에 대해서 코드 수준까지 분석하여 재사용성과 효율성에 대해서 컨설팅합니다
