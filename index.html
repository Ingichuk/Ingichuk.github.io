  • 👍 추천시스템 : 규모에 상관없이 추천 시스템 도입으로 CTR을 높여드립니다
  • 👍 GTP/AI : 경쟁사보다 빠르게 AI를 도입할 수 있도록 만들어 드립니다
  • 아키텍쳐 : 현재 사용자 규모에 적합하고, 확장이 용이한 아키텍쳐를 안내해드립니다
  • 데이터 관리 : 빅데이터를 위해서 각 기업 수준에 맞는 데이터 관리 방법을 알려드립니다
  • 보안 : 보안에 취약한 스타트업에 환경에서 최소한의 노력으로 최대의 보안을 유지할 수 있도록 합니다
  • DevOps : 최소한으로 갖춰야할 프로덕트 개발 방법론, 툴에 대해서 알려드립니다
  • iOS/Android : 네이티브 앱에 대해서 코드 수준까지 분석하여 재사용성과 효율성에 대해서 컨설팅합니다
