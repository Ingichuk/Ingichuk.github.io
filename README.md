# 인문과 기술의 척도, 인기척

인기척에는 수많은 인재들이 있습니다.

저희는 늘 새 프로젝트를 갈망하고 있습니다.

젊음을 갖춘 최고의 공학자, 철학자, 예술가들이 어우러져 청춘을 불태우고 있습니다.

큰 세상을 바꿀 조그마한 아이디어, 저희에게 주십시오.
